My name is Lucas, and I live in Rio. It's been just recently I got myself into the Front-end web development. Studying is like a Hobby for me and because of that I fell in love for that tech field and I am sure that every developer must be always up-to-date about new updates as well as trending technologies of the moment, and also I am sure that in the internet there are plenty of available contents allowing me to learn anything I want.

HTML me permite criar toda a parte esquelética do meu site e também criar o conteúdo incluso nele.

HTML allows me to create the whole website's body as well as creating the content within it.

CSS me permite trabalhar com vastos aspectos de um site, variando de estilização até responsividade. Ele me permite trabalhar com o estilo da página e seus efeitos e também ajustar a todas as resoluções.

CSS allows me to work with a big amount of a website's aspect, ranging from styling to responsiveness It allows me to work with the style of the page as well as its effects and also align it to any resolution.

Javascript torna a página mais interativa com o usuário. Assim consigo deixar o site mais dinâmico e interativo.

Javascript allows the web page to be more interactive with the user. This way I can make a more dynamic and interactive website as well.

PHP é uma linguagem de programação server-side, com ele eu consigo trabalhar com a parte do backend, assim eu posso trabalhar com banco de dados e consequentemente deixar o site mais dinâmico.

Bootstrap is a framework created by Twitter employees and it allows me to have more speed at creating my front-end projects.
