Me chamo Lucas, e moro no Rio. Há pouco tempo engressei na área de Desenvolvedor Front end. Estudar é praticamente um Hobby para mim e por este motivo que acabei me apaixonando pela área de tecnologia e sei que cada Desenvolvedor deve estar sempre informado de novas atualizações e tecnologias do momento, e sei que na Internet há uma ampla quantidade de conteúdos disponíveis fazendo com que eu consiga aprender qualquer coisa.

HTML me permite criar toda a parte esquelética do meu site e também criar o conteúdo incluso nele.

HTML allows me to create the whole website's body as well as creating the content within it.

CSS me permite trabalhar com vastos aspectos de um site, variando de estilização até responsividade. Ele me permite trabalhar com o estilo da página e seus efeitos e também ajustar a todas as resoluções.

CSS allows me to work with a big amount of a website's aspect, ranging from styling to responsiveness It allows me to work with the style of the page as well as its effects and also align it to any resolution.

Javascript torna a página mais interativa com o usuário. Assim consigo deixar o site mais dinâmico e interativo.

Javascript allows the web page to be more interactive with the user. This way I can make a more dynamic and interactive website as well.

React é um framework JavaScript criado pelo Facebook e que comecei a estudar há pouco tempo e ainda estou cobrindo a parte dos fundamentos. Ele me permitirá escrever códigos Javascript mais rápidos e de maior qualidade.

React is a Javascript framework created by Facebook and that I just started out to learn and I'm still covering the fundamental parts. It will allow me to write fast and better-quality javascript coding.
